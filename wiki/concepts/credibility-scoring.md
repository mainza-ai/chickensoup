# Credibility Scoring

The credibility scoring system for Project Chicken Soup defines how claims are scored based on evidence.

## Credibility Score

- **Range** — 0 to 1
- **High** — 0.8-1.0 (e.g., David Grusch testimony)
- **Medium** — 0.5-0.8 (e.g., Bob Lazar claims)
- **Low** — 0.0-0.5 (e.g., speculative claims)

## Factors

- **Source** — Credibility of the source
- **Evidence** — Strength of the evidence
- **Consensus** — Level of consensus
- **Time** — How long the claim has been around

## Integration

- **Neo4j** — Store credibility scores in Neo4j
- **Graph queries** — Query by credibility score
- **Dynamic** — Scores can be updated
