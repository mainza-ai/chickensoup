# Project Chicken Soup

Project Chicken Soup is a production-quality, local-first AI system that simulates and explores time travel using quantum computation, with a rich knowledge graph of UFO/Alien/Time Travel lore, and an AI agent that orchestrates everything.

## Overview

The project combines three layers of quantum computation with a knowledge graph and AI agents to create a system that can:

- Simulate time travel through spacetime
- Manipulate fields to enable movement through the field
- Navigate optimal paths through spacetime
- Maintain a rich knowledge graph of UFO/Alien lore
- Provide AI-powered interpretation of quantum results

## Architecture

See [[time-travel-machinery-architecture]] for detailed architecture.

The system uses **pydantic-graph** for core agent orchestration and **LangGraph** for complex sub-workflows, with a local-first LLM layer (oMLX, Ollama, LM Studio).

## Key Components

- **Spacetime Engine (Qiskit)** — Simulates the fabric of time
- **Field Manipulator (CUDA-Q)** — Manipulates the field of spacetime
- **AI Navigator (PennyLane)** — Finds optimal paths through spacetime
- **Knowledge Graph (Neo4j)** — Rich graph of UFO/Alien lore
- **Agent Framework (Pydantic AI)** — Multi-agent orchestration
- **API (FastAPI)** — REST API with OpenAPI specification
- **MCP Server (FastMCP)** — Model Context Protocol server
- **LLM Layer (oMLX/Ollama/LM Studio)** — Local-first LLM, auto-discovery

## Timeline Model

Many-Worlds. When you travel back in time, you don't change the past — you branch into a new timeline. The original timeline continues unchanged.
