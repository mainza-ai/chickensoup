# CUDA-Q

NVIDIA's open-source hybrid quantum-classical computing platform. Designed for GPU acceleration.

## Features

- **Hybrid quantum-classical** — Runs on both CPU and GPU
- **MLIR-based compiler** — nvq++ for quantum kernels
- **Multi-GPU workflows** — Scale large simulations
- **Density-matrix noise simulators** — Characterize noisy circuits
- **Direct QPU submission** — Unified API for multiple providers

## Why CUDA-Q for Time Travel

- Hybrid quantum-classical — connects AI to quantum
- GPU acceleration gives speed for large simulations
- Best for the neural-field model that finds time travel paths
- Unified API for CPU, GPU, and QPU
