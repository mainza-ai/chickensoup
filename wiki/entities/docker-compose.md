# docker-compose

docker-compose is a tool for defining and running multi-container Docker applications.

## Purpose

docker-compose uses a YAML file to define services, networks, and volumes for a multi-container application.

## Usage in Project Chicken Soup

- **Multi-container** — Run multiple containers (e.g., Neo4j, Redis, FastAPI)
- **Service orchestration** — Define service dependencies and startup order
- **Volume management** — Manage data volumes for persistent storage
- **Network management** — Define networks for inter-container communication
