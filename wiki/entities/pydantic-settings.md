# Pydantic Settings

Pydantic Settings is a configuration management tool for Pydantic models.

## Purpose

Pydantic Settings provides type-safe configuration loading from environment variables, files, and other sources.

## Usage in Project Chicken Soup

- **Configuration loading** — Load configuration from `.env` files, environment variables, and files
- **Type safety** — Configuration values are validated against Pydantic models
- **Default values** — Default values for configuration options
- **Nested configuration** — Support for nested configuration structures
