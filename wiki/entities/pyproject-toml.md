# pyproject.toml

pyproject.toml is the build configuration file for Python projects.

## Purpose

pyproject.toml defines the build system, dependencies, and metadata for a Python project.

## Usage in Project Chicken Soup

- **Build system** — Define the build system (e.g., setuptools, poetry)
- **Dependencies** — Define project dependencies
- **Metadata** — Define project metadata (name, version, description)
- **Configuration** — Define tool configuration (e.g., pytest, black, mypy)
