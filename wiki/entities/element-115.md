# Element 115

An exotic element (Moscovium) that produces gravity waves for propulsion, according to Bob Lazar.

## Lazar's Claim

Bob Lazar claims that Element 115 is the key to the propulsion system of alien craft. He says it produces gravity waves that allow the craft to move through space without conventional propulsion.

## Key Claims

- **Element 115** — A newly discovered element (Moscovium)
- **Gravity waves** — Produce propulsion for alien craft
- **Circular accelerator** — Used to create Element 115
- **Propulsion** — The craft are propelled by gravity waves generated by Element 115

## Controversy

- Some details have been questioned
- His claims about Element 115 are scientifically questionable
- The element has been discovered, but its role in propulsion is uncertain
