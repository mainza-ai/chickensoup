# Element 115

Element 115 (moscovium) is the exotic element that Bob Lazar claims powers alien craft.

## Bob Lazar's Claims

- **Recovery** — Lazar claims that Element 115 was recovered from a crashed UFO in the late 1940s or early 1950s
- **Properties** — Element 115 generates intense gravitational fields when energized, which powers the craft's propulsion system
- **Discovery** — Lazar claims he helped discover the element's properties while working at S-4
- **Connection to antigravity** — The gravitational field generated by Element 115 is thought to be the mechanism behind the craft's antigravity propulsion
- **7.46 Hz resonance** — Element 115 is claimed to resonate at 7.46 Hz, making this frequency a key parameter for understanding UAP propulsion

## Connection to Other Theories

- **T.T. Brown** — Thomas Townsend Brown's work on the Biefeld-Brown effect suggests that electric fields can manipulate gravity, providing a mechanism for how Element 115 generates thrust
- **Field manipulation** — Element 115 is a specific example of the broader field manipulation theory
- **Antigravity** — Element 115 is the specific mechanism by which antigravity is achieved in UAPs
- **7.46 Hz** — The frequency at which Element 115 resonates is believed to be 7.46 Hz, connecting it to the Schumann resonance and the Earth's electromagnetic field
