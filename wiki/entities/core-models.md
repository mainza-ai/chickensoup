# Core Models

Phase 1, item #4 in the project spec. Core models define the data structures used throughout the time travel system.

## Purpose

The core models are Pydantic schemas that define entities, relationships, timelines, and queries. They form the shared contract between the knowledge graph, the API, and the agents.
