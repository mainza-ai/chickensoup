# PennyLane

A cross-platform, differentiable quantum programming framework for quantum machine learning.

## Overview

PennyLane is developed by Xanadu and is purpose-built for quantum machine learning. It provides differentiable quantum programming, which means you can train quantum circuits like neural networks.

## Key Features

- **Differentiable quantum programming** — Train quantum circuits with gradient descent
- **QML frameworks** — Integrates with PyTorch, TensorFlow, JAX
- **Hardware agnostic** — Runs on IonQ, Rigetti, IBM, D-Wave, and more
- **CUDA support** — GPU-native quantum computation via CuQuantum
- **Hybrid models** — Combine classical and quantum layers seamlessly
